# Explainable Machine Learning Models for Diabetes Risk Prediction

## Overview
This project presents an explainable machine learning approach for predicting the risk of Type-2 diabetes. The focus is on developing models that are both accurate and interpretable, enabling better understanding of the factors influencing predictions.

## Key Contributions
- Developed machine learning models for diabetes risk prediction  
- Applied explainable AI techniques to interpret model outputs  
- Addressed class imbalance in healthcare datasets  
- Evaluated models using robust validation methods  

## Methods
- **Models:** Ensemble-based machine learning approaches  
- **Data Processing:** Handling class imbalance  
- **Explainability:** Feature importance and interpretability methods  

## Results
- Achieved strong predictive performance  
- Improved transparency in model decision-making  
- Enabled better understanding of key risk factors  

## Publication
This work was peer-reviewed, accepted, presented, and published at **IMCOM 2026 (IEEE)**.

## Purpose
This project contributes to the development of trustworthy AI systems in healthcare, supporting informed decision-making and early risk identification.
